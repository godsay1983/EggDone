# EggDone 手动回归清单

每次准备发布 Windows 安装包前执行本清单。测试时至少覆盖一个全新用户环境和一个已有数据库环境。

安装、覆盖升级、降级拦截和静默卸载可先使用 `scripts/verify-windows-installer.ps1` 自动检查；涉及应用启动、用户数据和系统集成的项目仍需手动验证。

## 安装与启动

- [ ] NSIS 安装包可由普通用户完成安装，无需管理员权限。
- [ ] 首次启动只显示一个托盘图标，不弹出普通主窗口。
- [ ] 重复启动不会出现第二个进程或第二个托盘图标。
- [ ] 托盘悬停提示显示精确的未完成数和总任务数。
- [ ] 托盘角标在浅色和深色任务栏上可辨认。

## 托盘与窗口

- [ ] 左键托盘图标可打开和隐藏面板。
- [ ] 托盘右键菜单的打开、新增、关于和退出均可用。
- [ ] 面板失焦和关闭按钮只隐藏窗口，不结束进程。
- [ ] 全局快捷键可切换面板并聚焦新增输入框。
- [ ] 在 Windows 100%、125%、150%、200% 缩放下验证位置和清晰度。
- [ ] 多显示器包含负坐标和混合 DPI 时，面板不超出工作区。

## Todo 操作

- [ ] 回车新增任务，空标题不会创建。
- [ ] 双击编辑，Enter、Escape 和点击空白区域行为正确。
- [ ] 完成、取消完成、删除、撤销和清除已完成可持久化。
- [ ] 拖动排序预览、高亮和最终顺序正确。
- [ ] 连续快速操作不会重复创建或错误覆盖任务。
- [ ] 托盘角标在每次任务变化后立即更新。

## 便签操作

- [ ] 切换到“便签”后可新建、编辑、置顶、换色、删除和撤销删除。
- [ ] 空白新便签直接返回时不写入数据库，也不出现空白卡片。
- [ ] 输入标题或正文后等待自动保存，隐藏面板或切换回 Todo 再返回时内容完整。
- [ ] 快速连续输入后立即关闭编辑器，最后一次输入仍被强制保存。
- [ ] 搜索同时匹配标题和正文，清空搜索后恢复置顶优先、最近修改优先的顺序。
- [ ] 亮色、暗色及长文本下，便签卡片、颜色圆点和编辑器操作区无重叠或低对比度。

## 数据可靠性

- [ ] 旧数据库可自动迁移且任务、完成状态和排序不丢失。
- [ ] JSON 导出后可在全新数据库中导入，分组、任务和便签均完整恢复。
- [ ] 缺少 `notes` 字段的旧 JSON 可导入且不影响本地便签。
- [ ] 无效、高版本或重复 UUID 的 JSON 被拒绝。
- [ ] SQLite 手动备份可被 SQLite 工具正常打开。
- [ ] 升级安装后应用数据和设置保持不变。
- [ ] 卸载应用后按发布策略确认用户数据是否保留。

## 同步

- [ ] MinIO HTTP 需明确确认明文风险后才能保存。
- [ ] MinIO HTTPS 和至少一种标准 S3 服务可测试连接。
- [ ] 首次同步可创建远端对象。
- [ ] 两台设备新增、编辑、完成、排序和删除后可正确合并。
- [ ] 断网不影响本地 Todo，状态显示为离线并有限重试。
- [ ] 持续 ETag 冲突停止上传且本地数据仍保留。
- [ ] Access Key 和 Secret Key 不出现在 SQLite、日志或错误文本中。
- [ ] 首次同步分别创建 Todo Object Key 和自动推导的 `notes.json` Object Key。
- [ ] 设置页展示的便签 Object Key 与 Todo Object Key 同目录，修改 Todo Object Key 后立即重新推导。
- [ ] 桌面创建便签、鸿蒙编辑并换色、桌面删除后，两端最终内容和删除状态一致。
- [ ] Todo 同步成功但便签同步失败时，不误报“已同步”，本地便签继续保留且下次可重试。

## 主题与无障碍

- [ ] 亮色、暗色主题均无低对比度或文字溢出。
- [ ] 系统减少动态效果开启时，拖动和列表动画被缩减。
- [ ] 键盘可完成新增、编辑、设置关闭和主要按钮操作。

## 语言与长文案

- [ ] 分别选择简体中文、English 和跟随系统，主面板无需重启即可切换且选择会持久化。
- [ ] Windows 系统语言在中文与英文间切换后，“跟随系统”会更新主面板、托盘菜单、通知和专注窗口。
- [ ] 默认窗口与最窄窗口下检查 Todo、便签、设置、数据管理和关于页，英文长文案可换行且按钮不重叠。
- [ ] 亮色与暗色主题下，中英文正文、副文本、禁用态和错误提示均满足可读对比度。
- [ ] 中英文快捷新增产生相同日期、提醒、重复、分组和重要字段；未知词与用户标题大小写保持不变。
- [ ] 托盘菜单、系统提醒、独立专注窗口和错误建议不残留另一种语言或底层异常堆栈。
- [ ] 运行 `pnpm i18n:check`、伪本地化测试和完整 `pnpm release:check`，结果均通过。

## 升级与卸载

- [ ] 从上一版本覆盖安装到当前版本成功。
- [ ] 尝试安装更旧版本时被安装器阻止。
- [ ] 卸载项显示 EggDone 名称、版本和正确图标。
- [ ] 卸载后应用进程、托盘图标和开机启动项均被移除。
