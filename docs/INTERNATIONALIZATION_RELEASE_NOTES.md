# EggDone 1.0.6 国际化发布说明

## 本次更新

- 新增简体中文、English 和跟随系统三种界面语言模式。
- 主面板、任务、便签、设置、数据管理、专注窗口、托盘菜单与系统通知使用统一翻译目录。
- 快捷新增同时识别中英文日期、时间、提醒、重复、重要标记和已有分组。
- 数据库、同步、附件、提醒和专注错误使用稳定错误码，并按当前语言显示简短说明与处理建议。
- 关于页版本直接读取 `package.json`，当前显示版本为 `1.0.6`。

## 自动门禁

- `pnpm i18n:check` 检查中英文字典缺失键、多余键、占位符和 Svelte 模板中文硬编码。
- 单元测试使用两端一致的 `docs/fixtures/quick-add-i18n-v1.json` 验证快捷新增语义。
- 伪本地化测试扩展全部英文文案并校验占位符，不发布伪语言资源。
- `pnpm release:check` 串联国际化检查、Svelte 检查、前端测试、生产构建和 Rust 检查。

## 发布前人工验收

按 `docs/MANUAL_REGRESSION.md` 的“语言与长文案”清单，在 Windows 中文、英文、跟随系统、亮暗主题、默认与窄窗口组合下验收。托盘、通知和独立专注窗口必须使用真实 Tauri 应用验证。
